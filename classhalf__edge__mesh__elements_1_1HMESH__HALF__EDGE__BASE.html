◆ PrevHalfEdgeAroundFromVertex()

half_edge_mesh_elements.HMESH_HALF_EDGE_BASE.PrevHalfEdgeAroundFromVertex (   self)

Return previous half edge around from vertex.

  • Returns PrevHalfEdgeInCell() if PrevHalfEdgeInCell() is a boundary half edge.
  • NextHalfEdgeAroundFromVertex() is not defined, since PrevHalfEdgeAroundFromVertex() should always be used in moving around a vertex.

◆ PrevHalfEdgeAroundVertex()

half_edge_mesh_elements.HMESH_HALF_EDGE_BASE.PrevHalfEdgeAroundVertex (   self,
  iv 
)

Returns previous half edge around vertex iv.

  • Returns PrevHalfEdgeInCell() if PrevHalfEdgeInCell() is a boundary half edge.
  • Note: If iv == ToVertexIndex(), returns NextHalfEdgeInCell().NextHalfEdgeAroundEdge() so that repeated calls to PrevHalfEdgeAroundVertex() move in a consistent direction.
